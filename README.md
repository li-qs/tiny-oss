# My OSS

一个针对图片的、轻量的文件存储服务，可作为图床使用。

## API 鉴权

```
Header: X-API-Token=<api_token>
```

或者 Query parameters:

```
?api_token=<api_token>
```

## API

### Web 静态文件服务

访客可通过 URL 查看图片

```
[GET] /i/<filename>
```

Query:

| 参数  | 类型   | 必填 | 说明                                     | 示例           |
| ----- | ------ | ---- | ---------------------------------------- | -------------- |
| token | string | 否   | 如果是私有文件，需要提供 token，否则不用 | hjk32g4hkj23h4 |

### 上传文件

后台可通过此接口上传文件

```
[POST] /api/upload
```

**需要进行 API 鉴权**

Form：

| 参数     | 类型 | 必填 | 说明                                                            | 示例                       |
| -------- | ---- | ---- | --------------------------------------------------------------- | -------------------------- |
| file     | file | 是   | 图片文件                                                        | /Users/bob/images/temp.jpg |
| compress | int  | 否   | 是否要压缩图片质量。是：1，否：0                                | 1                          |
| quality  | int  | 否   | （如果要压缩）压缩质量。10-100                                  | 80                         |
| private  | int  | 否   | 是否私有（通过 URL 访问的时候需要带上 token）。私有：1，公开：0 | 0                          |

### 查看文件列表

后台可通过此接口分页查询已上传的文件

```
[GET] /api/files
```

**需要进行 API 鉴权**

Query:

| 参数       | 类型   | 必填 | 说明                                          | 示例 |
| ---------- | ------ | ---- | --------------------------------------------- | ---- |
| page       | int    | 否   | 页码                                          | 1    |
| page_size  | int    | 否   | 每页多少条数据，20-100                        | 20   |
| sort_by    | string | 否   | 排序依据，id、created_at、size、original_name | id   |
| sort_order | string | 否   | 排序方式，asc、desc                           | 1    |

### 删除指定文件

后台可通过此接口删除已上传的文件

```
[DELETE] /api/<id>
```

**需要进行 API 鉴权**
